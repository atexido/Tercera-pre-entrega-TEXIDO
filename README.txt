Myapp

Este es un proyecto creado para administrar la lista de productos de una tienda en línea. A continuación, se explicará la funcionalidad que se ofrece:

Lista de productos
En la página principal se muestra la lista completa de productos disponibles en la tienda. Para cada producto se muestra su nombre, descripción, precio y un enlace para ver detalles.

Detalles del producto
Al hacer click en el enlace para ver detalles de un producto, se muestra su nombre, descripción y precio.

Agregar productos
En la página "Agregar Producto" se pueden agregar nuevos productos a la lista. Se debe completar un formulario que incluye campos como el nombre, descripción, precio e imagen del producto.

Buscar productos
En la página "Buscar Productos" se puede buscar un producto en particular ingresando su nombre o descripción. Se mostrará una lista con los productos que coinciden con los términos de búsqueda.

¡Gracias por utilizar nuestro sistema de administración de productos!
